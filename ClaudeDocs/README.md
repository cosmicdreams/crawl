## üìã Project Context

This is a Vue.js-based design token crawler that extracts design tokens from CSS and generates comprehensive reports. The project includes:

- **Core Application**: Design token extraction and analysis
- **Test Suite**: Comprehensive unit and E2E testing
- **Storybook**: Component documentation and testing
- **Multiple Output Formats**: JSON, CSV, HTML reports

## üîß Key Technologies

- Vue.js 3 with TypeScript
- Vite build system
- Vitest for testing
- Playwright for E2E testing
- Storybook for component documentation
